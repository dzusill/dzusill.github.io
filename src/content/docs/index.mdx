## Welcome 👋

Glad you're here! Whether you're spinning up your first plugin or fine-tuning a packed server, you're in the right place. Every dzusill plugin is documented the same friendly way, so once you know one, you know them all:

- 🚀 **Getting Started** — install it and have it running in minutes.
- ✨ **Features** — every feature explained, with examples.
- ⚙️ **Configuration** — every config key, what it does, and sensible defaults.
- 📖 **Reference** — commands, permissions, placeholders and FAQs at a glance.

New around here? Start with **🛠️ DzusillCore** — the framework most of the others are built on — or just jump straight to whatever plugin you're running below. 🔍 The search bar (top right) finds anything across every plugin.

## The plugins

| Plugin | What it does |
| --- | --- |
| [🛠️ DzusillCore](/plugins/dzusillcore/) | Reusable Paper plugin template framework |
| [📊 dStattrack](/plugins/dstattrack/) | Item stat tracker built on the DzusillCore framework |
| [🧭 WarpGUI](/plugins/warpgui/) | Warp menu with categories, extra locations & per-warp permissions |
| [🏠 dHomeGUI](/plugins/dhomegui/) | Homes GUI with limits, warmups & world rules (Folia-ready) |
| [💰 dBloodMoney](/plugins/dbloodmoney/) | Kill a player, take their money |
| [🔔 ToolsNotifier](/plugins/toolsnotifier/) | Low-durability tool & armor notifications (Free + PRO) |
| [💀 dDeathPenalty](/plugins/ddeathpenalty/) | Punishment-style death penalty (money / xp / item loss) |
| [🎰 dLottery](/plugins/dlottery/) | Server lottery with MySQL-backed rounds, history & stats |

Happy building! 🎉
