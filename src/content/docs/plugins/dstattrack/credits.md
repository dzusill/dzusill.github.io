**dStattrack** is developed by **DZUSILL**.

- Built on the [DzusillCore](https://github.com/dzusill/DzusillCore) framework (bundled).
- Item NBT storage via [NBTAPI](https://www.spigotmc.org/resources/nbt-api.7939/).
- Economy via [Vault](https://www.spigotmc.org/resources/vault.34315/).

A modern rebuild of the original 1.16.2 stattrack plugin, updated for Paper 1.21.1 (mace, hoe and shears tracking, the declarative command tree and GUI, and graceful degradation when soft dependencies are missing).

Found a bug or have a request? Reach out through the resource page where you downloaded dStattrack.
