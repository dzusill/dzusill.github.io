**DRotatingShop** is developed by **dzusill**.

- Built on the [DzusillCore](https://github.com/dzusill/DzusillCore) framework.
- Required: [Vault](https://www.spigotmc.org/resources/vault.34315/) + any Vault economy plugin.
- Optional: [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) for the countdown [placeholder](/plugins/drotatingshop/placeholders/).

A rotating GUI shop: a timed pool of random items, buy-only Vault pricing, global stock and per-rotation per-player limits, Base64-serialised custom items, and restart-safe persistence — all on DzusillCore's declarative command and menu system.

Found a bug or have a request? Reach out through the resource page where you downloaded DRotatingShop.
