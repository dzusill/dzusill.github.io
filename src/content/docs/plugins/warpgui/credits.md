**WarpGUI** is developed by **DZUSILL**.

- Built on the [DzusillCore](https://github.com/dzusill/DzusillCore) framework.
- Optional integrations: [Vault](https://www.spigotmc.org/resources/vault.34315/), [Essentials](https://essentialsx.net/), [CMI](https://www.zrips.net/cmi/), [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/).

A modern rebuild of the original WarpGUI: a declarative command tree and menu system, fixed warp-limit and promotion-rotation bugs, random multi-location teleports, MySQL/PostgreSQL storage and graceful soft-dependency loading.

Found a bug or have a request? Reach out through the resource page where you downloaded WarpGUI.
