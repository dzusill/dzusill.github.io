| Requirement | Version | Notes |
|---|---|---|
| Server software | Paper / Spigot **1.21.x** | `api-version` is 1.17, so it runs on 1.17+; durability handling is tuned for 1.21. |
| Java | **21** | |
| [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/) | optional | Enables the `%toolsnotifier_…%` [placeholders](/plugins/toolsnotifier/placeholders/). |

ToolsNotifier is **standalone** — it doesn't depend on any framework. PlaceholderAPI is the only (soft) dependency, and only for placeholders.

## Editions

There are two downloads:

- **ToolsNotifier** (free) — all notification features.
- **ToolsNotifierPro** — everything in free **plus** the preferences GUI, inventory scan and admin monitor.

Install one or the other, not both. See [Free vs PRO](/plugins/toolsnotifier/getting-started/free-vs-pro/).
