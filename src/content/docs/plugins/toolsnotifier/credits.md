**ToolsNotifier** is developed by **DZUSILL**.

- Standalone Paper/Spigot plugin — no framework dependency.
- Optional integration: [PlaceholderAPI](https://www.spigotmc.org/resources/placeholderapi.6245/).
- Config comments are preserved across reloads via a CommentedConfiguration wrapper.

Recent highlights (v1.4): per-category thresholds, per-item overrides, `/tn reload`, accurate 1.21 durability handling, and hardened event handling. The PRO edition adds the per-player preferences GUI, periodic inventory scan and admin gear monitor.
