**dLottery** is developed by **DZUSILL**.

- Built on the [DzusillCore](https://github.com/dzusill/DzusillCore) framework (bundled).
- Economy via [Vault](https://www.spigotmc.org/resources/vault.34315/).
- Persistence via MySQL / MariaDB.

A modern rebuild of the original lottery plugin: weighted draws, a live auto-refreshing GUI, transactional ticket purchases (charge-then-persist with automatic refund on failure), offline payouts, a sortable leaderboard, and full MySQL persistence.

Found a bug or have a request? Reach out through the resource page where you downloaded dLottery.
